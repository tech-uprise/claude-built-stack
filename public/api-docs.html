REST API Endpoints
Base URL: http://localhost:3000/api

System Endpoints

GET /api/health
System health check endpoint. Returns server status and current timestamp for monitoring and uptime verification.
Response Example
{ "status": "ok", "timestamp": "2025-12-10T17:50:00.000Z" }
GET /api/test-db
Database connection test endpoint. Verifies PostgreSQL connectivity and returns current database timestamp.
Response Example
{ "status": "success", "message": "Database connection successful", "data": { "current_time": "2025-12-10T17:50:00.000Z" } }

User Management

GET /api/users
Retrieve all registered users from the database. Returns user count and complete user list ordered by registration date.
Response Example
{ "status": "success", "count": 2, "users": [ { "id": 1, "name": "John Doe", "email": "john@example.com", "created_at": "2025-12-10T17:50:00.000Z" } ] }
POST /api/users
Create a new user record. Requires name and email in request body. Email must be unique.
Request Body
{ "name": "John Doe", "email": "john@example.com" }
Response Example
{ "status": "success", "message": "User created successfully", "user": { "id": 1, "name": "John Doe", "email": "john@example.com", "created_at": "2025-12-10T17:50:00.000Z" } }
DELETE /api/users/:id
Delete a specific user by ID. Returns deleted user information upon successful deletion.
URL Parameters
id (integer) - User ID to delete
Response Example
{ "status": "success", "message": "User deleted successfully", "user": { "id": 1, "name": "John Doe", "email": "john@example.com" } }
